# UrbanThreads - Clothing Brand E-commerce Website

## Setup Instructions
1. Create the folder structure as shown above
2. Copy all code into respective files
3. Serve using any local server (Live Server extension recommended)
4. Open index.html to start browsing

## Features Implemented
✅ Fully responsive design (mobile, tablet, desktop)
✅ Sticky navbar with search
✅ Hero slider with auto/manual navigation
✅ Product listing with categories
✅ Cart functionality with localStorage
✅ Wishlist with localStorage
✅ Authentication modal
✅ Toast notifications
✅ Back to top button
✅ GSAP scroll animations
✅ Product cards with hover effects

## Pages Structure
- index.html - Home page with slider, categories, featured products
- products.html - Product listing with filters & sorting
- product-detail.html - Individual product view
- cart.html - Shopping cart management
- wishlist.html - Saved items
- checkout.html - Order placement

## Technologies Used
- HTML5, Tailwind CSS, JavaScript
- GSAP for animations
- Font Awesome icons
- LocalStorage for data persistence
